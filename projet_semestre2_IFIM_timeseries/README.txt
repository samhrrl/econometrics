Ce projet économétrique traitant des investissements des administrations publiques se constitue des repertoires suivants:

dataset_bea: fichier excel tel que je les ai telechargé sur le site de la bea

versions_projet: contient toutes les versions du projet

econometrics_R_project.R: est le fichier du projet sous R

presentation_canva_econometrics: support de présentation orale

rapport_R_econometrics.docx: rapport du projet au format word

rapport_R_econometrics.pdf: rapport du projet au format pdf

README: information que vous lisez

receipt.xlsx: le fichier utilisé dans le cadre du projet passé sous l'algorithme de Python pour organiser les données

script_python_nettoyage_donees : programme Python qui nettoie les données et les rends exploitables
